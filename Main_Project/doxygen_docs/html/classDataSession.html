Detailed Description

Stores a single run of data for one sensor, and exposes an API to add, clear, and retrieve stored data.

DataSession is a simple container that stores the collected data for a single sensor. Each sensor has its own DataSession object to hold all the values that we read from it. This allows a direct associate of data to each Sensor.

Note
There is no class ownership relationship between Sensor and DataSession. A DataSession is associated in name only with a specific sensor.
