\doxysubsection{Detailed Description}
Main GUI page used to launch into projects. 

\doxylink{classMainFrame}{Main\+Frame} class is the main application window. It contains:\+


\begin{DoxyItemize}
\item A toolbar with features like Start, Stop, Collect, \doxylink{classSensor}{Sensor} and Graph, that are disabled unless a project is open, and where actions like start/\+ stop/\+collect/\+graph won\textquotesingle{}t run unless a sensor is selected.
\item A sidebar on the left with New Project, Load Project, Connect, Sensors, and other user options.
\item A central Display window for showing \doxylink{classProjectPanel}{Project\+Panel} instances.
\end{DoxyItemize}

Each \doxylink{classProjectPanel}{Project\+Panel} instance is wrapped into a wx\+AUINotebook with switchable tabs.
