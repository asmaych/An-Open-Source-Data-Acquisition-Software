\doxysubsection{Detailed Description}
GUI object that encapsulates all \doxylink{classToolbar}{Toolbar} controls into one class. 

The \doxylink{classToolbar}{Toolbar} currently includes controls for:\+
\begin{DoxyItemize}
\item Start/\+\+Stop \doxylink{classRun}{Run}
\item Table Display Toggle
\item Graph Display Toggle
\item Reset \doxylink{classRun}{Run}
\item On Demand Toggle
\item Export Data
\item Project\+Config Launcher
\end{DoxyItemize}

Callbacks are used extensively in this class to affect the in-\/focus \doxylink{classProjectPanel}{Project\+Panel} via \doxylink{classMainFrame}{Main\+Frame}
