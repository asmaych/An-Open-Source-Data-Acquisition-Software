\doxysubsection{Detailed Description}
A lightweight helper class for the different methods of recording sensor values. 

\doxylink{classDataCollector}{Data\+Collector} is a lightweight helper for the 2 different methods of collecting that we have:\+
\begin{DoxyEnumerate}
\item continuous-\/collect that continuously appends incoming sensor readings into the current session until stopped
\item collect-\/last-\/value that appends only the last set of values received once per invocation
\end{DoxyEnumerate}

This class provides two methods that the GUI can call. it does not own sessions.
