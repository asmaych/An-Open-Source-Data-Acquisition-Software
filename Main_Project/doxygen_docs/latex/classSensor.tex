\doxysubsection{Detailed Description}
Virtual sensor used to reflect the real-\/world state of a single physical sensor. 

For each sensor connected to the microcontroller, there will be one instance of \doxylink{classSensor}{Sensor}. Each sensor has its own set of readings, and exposes a public API that allows readings to be set, and retrieved from the \doxylink{classSensor}{Sensor}.

Additionally, each \doxylink{classSensor}{Sensor} uses its own \doxylink{classCalibrator}{Calibrator} object, which modifies the behavior of the get\+Reading() method according to user-\/defined calibrations. If no calibration is specified, raw values are returned. Having each sensor own its own calibration allows for per-\/sensor calibration.
