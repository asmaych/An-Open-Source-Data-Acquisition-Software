\doxysubsection{Detailed Description}
GUI that provides users with a top-\/level view of all \doxylink{classSensor}{Sensor} Objects and interfaces to modify them. 

This is a dialog that shows users a list of all \doxylink{classSensor}{Sensor} objects currently configured in the project. It displays sensors by name in a table, and above that table, it displays buttons that allow the user to add a new \doxylink{classSensor}{Sensor}, remove or calibrate a selected \doxylink{classSensor}{Sensor}, and load a sensor from the sensor database.
