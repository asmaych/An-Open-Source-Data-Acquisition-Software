\doxysubsection{Detailed Description}
GUI object that displays a graph based on vectorized datapoints. 

Graphwindow is a class that displays a graph of the collected sensor data. each \doxylink{classGraphWindow}{Graph\+Window} is tied to a \doxylink{classDataSession}{Data\+Session} and can display/\+visualize its values.
