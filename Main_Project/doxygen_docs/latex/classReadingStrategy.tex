\doxysubsection{Detailed Description}
Interface for implementation of \doxylink{classReadingStrategy}{Reading\+Strategy} Strategy Pattern. 

This class has no implementation. It is an abstract class that is only used to construct one of many specific implementations. It allows the user to select one of three forms of sensor reading:\+ Raw, Voltage, and Mapped. By using the Strategy Pattern, runtime Polymorphism can be used to change which type of reading is returned from a \doxylink{classSensor}{Sensor} object\textquotesingle{}s get\+Reading() method.
