\doxysubsection{Detailed Description}
GUI object used to allow user entry of raw-\/mapped pairs into a table for use in an \doxylink{classInterpolator}{Interpolator}. 

When this dialog is created, it loads a table of raw-\/mapped pairs, that the user can add to or modify. These values are used to create a vector of \doxylink{structCalibrationPoint}{Calibration\+Point} objects, which are loaded into an \doxylink{classInterpolator}{Interpolator} for use in a specific \doxylink{classSensor}{Sensor} selected earlier by the user.

If there is no current calibration configured for the selected \doxylink{classSensor}{Sensor}, the table will be blank. Otherwise, the calibration table will be loaded with the current values.
