\doxysubsection{Detailed Description}
Class used to represent the global database of \doxylink{classSensor}{Sensor} objects, along with their configurations. 

\doxylink{classSensorDatabase}{Sensor\+Database} is the class responsible for saving and loading sensors to and from an SQLite database file. We are using the database to store the sensors, because without it, they disappear as soon as the app closes. This class creates a database file if it doesn\textquotesingle{}t exist, and a sensors table that will be saved in the db and loaded back into memory.
