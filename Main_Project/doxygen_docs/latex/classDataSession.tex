\doxysubsection{Detailed Description}
Stores a single run of data for one sensor, and exposes an API to add, clear, and retrieve stored data. 

\doxylink{classDataSession}{Data\+Session} is a simple container that stores the collected data for a single sensor. Each sensor has its own \doxylink{classDataSession}{Data\+Session} object to hold all the values that we read from it. This allows a direct associate of data to each \doxylink{classSensor}{Sensor}.

\begin{DoxyNote}{Note}
There is no class ownership relationship between \doxylink{classSensor}{Sensor} and \doxylink{classDataSession}{Data\+Session}. A \doxylink{classDataSession}{Data\+Session} is associated in name only with a specific sensor. 
\end{DoxyNote}
