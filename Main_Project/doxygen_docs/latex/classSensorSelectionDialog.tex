\doxysubsection{Detailed Description}
GUI object that allows user to select active sensors. 

This is a dialog window that allows the user to select which sensors to collect from in a \doxylink{classRun}{Run}. It displays a checklist of all sensors in the project. It returns the selected sensor indexes.
