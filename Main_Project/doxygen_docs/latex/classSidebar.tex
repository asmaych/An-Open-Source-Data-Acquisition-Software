\doxysubsection{Detailed Description}
GUI object that encapsulates all \doxylink{classSidebar}{Sidebar} controls into one class. 

The \doxylink{classSidebar}{Sidebar} currently includes controls for:\+
\begin{DoxyItemize}
\item New Project
\item Load Project (in progress)
\item Connect (to a microcontroller)
\item Sensors (launches \doxylink{classSensorConfigDialog}{Sensor\+Config\+Dialog}) 
\end{DoxyItemize}
